Справочник HTML-тегов для веб-дизайна

Полный список HTML-тегов, разделенных по категориям

Структура документа 9 тегов
<!DOCTYPE html>
Объявление типа документа (для HTML5)
<html>
Корневой элемент HTML-документа
<head>
Контейнер для метаданных (название, стили, скрипты, мета-теги)
<title>
Заголовок страницы (отображается во вкладке браузера)
<body>
Основное тело документа, где размещается всё видимое содержимое
<meta>
Предоставляет метаинформацию о странице (кодировка, описание, ключевые слова)
<link>
Подключает внешние ресурсы, чаще всего таблицы стилей (CSS)
<style>
Для написания CSS-стилей непосредственно в HTML-документе
<script>
Для подключения или написания JavaScript-кода
Семантические теги макета 10 тегов
<header>
"Шапка" сайта или раздела
<nav>
Раздел навигации по сайту
<main>
Основное, уникальное содержимое страницы
<article>
Самостоятельная часть контента (пост в блоге, статья, карточка товара)
<section>
Тематическая группировка контента, обычно с заголовком
<aside>
Боковая панель, косвенно связанная с основным контентом (сайдбар)
<footer>
"Подвал" сайта или раздела
<figure>
Автономный контент (иллюстрация, диаграмма, фото)
<figcaption>
Подпись для <figure>
Заголовки и текстовое содержимое 16 тегов
<h1>-<h6>
Заголовки разного уровня (от самого важного h1 до наименее важного h6)
<p>
Абзац текста
<span>
Универсальный строчный контейнер для стилизации части текста
<div>
Универсальный блочный контейнер для группировки элементов
<br>
Перенос строки
<hr>
Горизонтальная линия (тематический раздел)
<pre>
Выводит текст с сохранением форматирования (пробелы, переносы)
<blockquote>
Блок для длинных цитат
<q>
Короткая строчная цитата
<cite>
Цитирование источника (название книги, статьи)
<code>
Фрагмент программного кода
<time>
Для обозначения даты и/или времени
Списки 6 тегов
<ul>
Маркированный (ненумерованный) список
<ol>
Нумерованный список
<li>
Элемент списка (для <ul> и <ol>)
<dl>
Список описаний (терминов и их определений)
<dt>
Термин в списке описаний
<dd>
Описание/определение термина в <dl>
Ссылки и навигация 1 тег
<a>
Гиперссылка. Главный атрибут: href="URL"
Изображения и мультимедиа 10 тегов
<img>
Для вставки изображений. Главные атрибуты: src="путь", alt="описание"
<picture>
Контейнер для нескольких источников изображения (адаптивные изображения)
<source>
Определяет медиа-ресурсы для <picture>, <audio>, <video>
<audio>
Для вставки аудиоконтента
<video>
Для вставки видео контента
<iframe>
Для встраивания другой HTML-страницы (например, видео с YouTube, карты)
<svg>
Контейнер для Scalable Vector Graphics (векторная графика)
<canvas>
Используется для рисования графики через JavaScript (игры, диаграммы)
Таблицы 8 тегов
<table>
Контейнер для таблицы
<caption>
Заголовок (название) таблицы
<thead>
Блок заголовка таблицы
<tbody>
Основное тело таблицы
<tfoot>
Подвал (итоги) таблицы
<tr>
Строка таблицы
<th>
Ячейка-заголовок
<td>
Ячейка с данными
Формы и элементы ввода 13 тегов
<form>
Контейнер для элементов формы. Атрибуты: action, method
<input>
Универсальное поле ввода (текст, пароль, радио, чекбокс, кнопка, дата и др.)
<label>
Подпись для элемента формы. Улучшает доступность
<textarea>
Многострочное текстовое поле
<select>
Выпадающий список
<option>
Пункт внутри <select>
<optgroup>
Группа пунктов внутри <select>
<button>
Кнопка
<fieldset>
Группирует связанные элементы формы
<legend>
Заголовок для <fieldset>
<datalist>
Предоставляет варианты для автодополнения <input>
<output>
Отображает результат вычисления
Интерактивные элементы 2 тега
<details>
Создает интерактивный виджет, который можно раскрыть/свернуть
<summary>
Заголовок для элемента <details>
